# RecordToTalk
record-to-textの会話録音版です。最初は検証のため機能を必要最小限とします。
単に会話を録音して画面表示するのみとし、ローカルストレージへの保存はなしとします。
文字起こしの間隔は設定画面でカスタマイズできます。record-to-textと同様、`Flutter for macOS`で動作します。
